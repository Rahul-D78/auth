# Authentication of users 

## Technologies used :
1. Node.JS -- Platform
2. ejs -- templating engine
3. MongoDB -- Database
4. Javascript -- Language
5. Passport -- authentication
6. Ouath2.0 -- google authentication

## Database setup :
I have setup my db in local but you can do the same in cloud also 

## Installation :
```$npm install```

## run :
```$npm start```
